IP Addresses Demonstration

Welcome to the CS2026 Lab IP addresses page. This page demonstrates different types of IP addresses used in networking.

IPv4 Loopback Address:

127.0.0.1

IPv6 Loopback Address:

::1

Private Network (Class A):

10.0.0.0 - 10.255.255.255

Private Network (Class B):

172.16.0.0 - 172.31.255.255

Private Network (Class C):

192.168.0.0 - 192.168.255.255

Default Gateway (Common):

192.168.1.1
