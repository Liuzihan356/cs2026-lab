IP Addresses Demonstration

IPv4 Loopback Address:
127.0.0.1

The loopback address is used to test network software without physically connecting to a network.

IPv6 Loopback Address:
::1

The IPv6 loopback address, equivalent to 127.0.0.1 in IPv4.

Private IP Range (Class C):
192.168.0.0 - 192.168.255.255

Common private IP range used in home and small office networks.

Private IP Range (Class A):
10.0.0.0 - 10.255.255.255

Large private IP range often used in corporate networks.

Default Gateway Example:
192.168.1.1

Common default gateway address for home routers.
