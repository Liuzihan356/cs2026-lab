# CS2026 Lab - Git and GitHub Pages

This is a lab project for CS2026 course, demonstrating Git, SSH, and GitHub Pages usage.

## Project Structure
- `docs/` - Website files for GitHub Pages deployment
- `docs/ipaddresses.html` - Main page showing IP address information
- `docs/404.html` - Custom 404 error page

## Deployment
This site is deployed using GitHub Pages.
