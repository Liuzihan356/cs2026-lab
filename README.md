# CS2026 Lab - Git and GitHub Pages

This is a lab project for CS2026 Computer Networks course, demonstrating Git, GitHub, and GitHub Pages deployment.

## Project Structure

- `docs/` - Documentation and website files
  - `ipaddresses.html` - IP addresses demonstration page
  - `index.html` - Main website page
  - `404.html` - Custom 404 error page

## Features

- SSH key authentication with GitHub
- Git version control
- GitHub Pages deployment
- Static website hosting
